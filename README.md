El desafio fue realizado con Flask + React + PostgreSQL no opté por redux porque solo complicaria el código siendo repitivo y no aportaría valor según lo investigado.
En la carpeta backend se encuentra la API construida con Python y Flask, más la conexión a la BD de PostgreSQL.
Quise mantener las credenciales a la BD en el backend, pero se deben ocultar por seguridad en un entorno profesional.
En la carpeta Frontend se encuentra la interfaz desarrollada con React
Es necesario tener instalado Python, Node.js y npm y PostgreSQL
